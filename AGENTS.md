# Always commit and push after every change

After completing any code change the user asks for, immediately `git add` the relevant files, create a commit with a clear message, and `git push` to `origin main`. Do not wait to be told. If a push would be destructive (force-push, rewriting shared history, etc.), pause and confirm first — but normal commits and pushes happen automatically.
